Command:
1.npm init -y (this will add package.json)
2.npm i -g npm@latest (update npm):for error:https://github.com/npm/cli/wiki/%22cb()-never-called%3F--I'm-having-the-same-problem!%22
3.npm i -D webpack or npm install --save-dev webpack (install node_modules link: https://webpack.js.org/guides/installation/)
4.npm install -D webpack-cli (install webpack cli)
5.npm install -D babel-loader @babel/core @babel/preset-env webpack (install babel loader: https://webpack.js.org/loaders/babel-loader/)
6.npm i -D @babel/plugin-proposal-class-properties  (plugin link: https://babeljs.io/docs/en/babel-plugin-proposal-class-properties)
